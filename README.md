# RGB-D 相机的深度传感器的姿势估计
## 运行环境要求
- MATLAB R2016及以上
- 能够打开 mj2 类型的视频

## 运行说明
- 从 [视频存放网址](https://pan.baidu.com/s/1pLeAog7) 下载视频，将"color.avi"和"depth.mj2"放至根目录
- 进入根目录
- 直接在Matla命令行窗口输入 main 命令，按回车即可
